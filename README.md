**_Description_**

This app is a website for a fine-dining restaurant named Folha. It serves as an online reference for customers to learn about the restaurant, find restaurant locations, view the menu, and read reviews.

**_Technologies Used_**

- PostgreSQL
- Express.js
- Vue.js
- Node
- Heroku
- Tailwind CSS
- Sequelize
